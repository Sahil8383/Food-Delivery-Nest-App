## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Setup Environment Variables

Create a .env file in the root directory and add the following variables

```bash
PORT=3000
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=postgres
DB_PASSWORD=********
DB_DATABASE=

// Optional
REDIS_HOST=localhost
REDIS_PORT=6379

```
